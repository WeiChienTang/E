## 建議新增功能

### 優先順序說明

| 標記 | 說明 |
|------|------|
| ⭐⭐⭐ | 高優先 — 法規需求或明顯功能缺口 |
| ⭐⭐ | 中優先 — 流程補強，提升操作效率 |
| ⭐ | 低優先 — 管理便利性，可依需求安排 |

---

## 一、高優先功能 ⭐⭐⭐

### 1. 電子發票整合（e-Invoice）

**背景**：台灣財政部規定，B2B 電子發票自 2025 年起全面強制，B2C 亦有雲端發票規範。

**建議功能**：
- 銷貨出貨後自動開立電子發票
- 發票作廢與折讓單
- 上傳財政部平台（透過加值中心 API）
- 與現有銷貨單、沖款單串接
- 發票查詢與補印

**影響模組**：銷貨、財務

---

### 2. 請假管理（Leave Management）

**背景**：薪資模組已有出勤彙總，但缺少請假申請與審核流程，目前出勤資料需手動輸入，效率低。

**建議功能**：
- 假別設定（特休、病假、事假、公假…）
- 員工請假申請單
- 主管線上審核（核准 / 駁回）
- 假期餘額自動計算（依勞基法累積）
- 核准後自動帶入出勤彙總，減少手動輸入

**影響模組**：人力資源、薪資管理

---

### 3. 固定資產管理（Fixed Assets）

**背景**：會計模組已有傳票與科目，但無法追蹤資產生命週期，折舊傳票目前需手動輸入。

**建議功能**：
- 資產登記（類別、購入日期、購入金額、耐用年限）
- 自動折舊計算（直線法 / 餘額遞減法）
- 每月自動產生折舊傳票
- 資產移轉（部門間調配）
- 資產報廢 / 出售處理
- 資產清冊報表

**影響模組**：會計、財務

---

## 二、中優先功能 ⭐⭐

### 4. 廠商詢價 / 比價（RFQ）

**背景**：現有採購流程從採購單開始，缺少前段詢價環節，多廠商比價需仰賴 Excel。

**建議功能**：
- 詢價單建立（可從需求或排程產生）
- 發送詢價給多家廠商
- 廠商報價回收與比較表
- 選定廠商後一鍵轉採購單
- 報價有效期追蹤

**影響模組**：採購、廠商

---

### 5. 生產工單（Work Orders）

**背景**：已有生產排程和 BOM，但缺少工單執行追蹤，無法即時掌握生產進度與用料實際狀況。

**建議功能**：
- 工單開立（依排程或手動）
- 工單狀態追蹤（待開工 / 進行中 / 完工）
- 投料領料與 BOM 對照
- 完工回報（產出數量、良品 / 不良品）
- 庫存自動更新（領料扣減、完工入庫）
- 工單成本計算

**影響模組**：商品（生產排程）、庫存（領料）

---

### 6. 售後服務 / 維修單（Service Orders）

**背景**：銷貨流程完整，但客戶報修後無追蹤機制，需透過其他系統或紙本記錄。

**建議功能**：
- 客戶報修單（關聯銷貨單或客戶）
- 保固期自動判斷
- 維修進度追蹤（待處理 / 維修中 / 完成）
- 維修費用記錄（保固內免費 / 自費）
- 備品用料紀錄（與庫存串接）
- 維修報表與統計

**影響模組**：銷貨、客戶、庫存

---

### 7. 操作稽核記錄（Audit Log）

**背景**：目前只有系統錯誤記錄，無法追蹤「誰在何時對哪筆資料做了什麼操作」，不符合一般企業內控需求。

**建議功能**：
- 自動記錄所有新增 / 修改 / 刪除操作
- 記錄操作人員、時間、異動前後值
- 支援依模組、使用者、時間區間篩選查詢
- 敏感操作加強記錄（核准、關帳、權限異動）
- 匯出稽核報表

**影響模組**：系統管理（新增「稽核記錄」子項目）

---

### 8. 業績目標管理（Sales Targets / KPI）

**背景**：銷貨圖表已有業績分析，但無法設定目標值與實績對照，缺少達成率管理。

**建議功能**：
- 設定年度 / 月度銷售目標（依業務員 / 客戶 / 商品）
- 目標 vs 實績比較報表
- 達成率視覺化（儀表板圖表）
- 預警通知（月底前達成率偏低）

**影響模組**：銷貨、首頁儀表板

---

## 三、低優先功能 ⭐

### 9. 費用報帳（Expense Claims）

**建議功能**：
- 員工費用申請（出差 / 交際 / 雜支）
- 費用類別設定
- 主管審核流程
- 核准後自動產生應付帳款或薪資扣抵
- 費用統計報表

**影響模組**：財務、薪資管理

---

### 10. 廠商評鑑（Supplier Evaluation）

**建議功能**：
- 定期（季 / 年）廠商評鑑表
- 評分項目設定（交期達成率、品質、價格、服務）
- 自動統計歷史採購數據（交期、退回率）
- 評鑑等級（A / B / C）與合格供應商名單
- 比較報表

**影響模組**：廠商、採購

---

### 11. 公告 / 佈告欄（Announcements）

**建議功能**：
- 系統內部公告發布
- 目標群組設定（全公司 / 特定部門）
- 已讀確認追蹤
- 到期自動下架
- 首頁儀表板公告顯示

**影響模組**：系統管理、首頁

---

### 12. 行事曆（Calendar）

**建議功能**：
- 整合式月曆視圖
- 跨模組事件顯示（交貨日、盤點日、薪資發放日、保養到期日）
- 自訂提醒事件
- 行事曆與生產排程整合

**影響模組**：跨模組整合

---

### 13. 批次匯入工具（Bulk Import）

**建議功能**：
- 標準 Excel 匯入範本下載
- 支援批次匯入商品、客戶、廠商、員工基本資料
- 匯入前驗證預覽（顯示錯誤行）
- 匯入結果摘要報告

**影響模組**：各主要模組

---

## 四、便利性工具

### 14. 匯率歷史記錄

**現況**：已有貨幣設定，但可能缺少歷史匯率追蹤。

**建議**：
- 每日匯率紀錄（手動或串接 API）
- 指定日期換算功能
- 跨幣別財務報表（依歷史匯率換算）

---

### 15. 儀表板進階小工具

**現況**：已有圖表和捷徑小工具。

**建議新增小工具**：
- 「待處理事項」彙總（待審核採購單、待開立發票、即將到期請假…）
- 「今日出貨」快速摘要
- 「庫存不足警示」（低於安全庫存的商品清單）
- 「本月業績進度條」

---

## 五、建議開發順序

```
第一階段（合規 & 核心缺口）
├── 電子發票整合
├── 請假管理
└── 固定資產管理

第二階段（流程補強）
├── 廠商詢價 / 比價
├── 生產工單
└── 操作稽核記錄

第三階段（管理效率）
├── 售後服務 / 維修單
├── 業績目標管理
└── 費用報帳

第四階段（便利性工具）
├── 廠商評鑑
├── 公告 / 佈告欄
├── 行事曆
└── 批次匯入工具
```

## 六、各功能影響模組關係圖

```
請假管理 ──────────────► 薪資（出勤彙總自動帶入）
廠商詢價 ─────────────► 採購單（詢價轉採購）
生產工單 ──┬───────────► 庫存（領料扣減、完工入庫）
           └───────────► 生產排程（工單來源）
電子發票 ──────────────► 銷貨單 + 財務沖款
固定資產 ──────────────► 會計傳票（折舊自動產生）
售後服務 ──┬───────────► 客戶（報修來源）
           └───────────► 庫存（備品用料）
```
